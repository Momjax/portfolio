RP3 - Jeu d'Arène Multijoueur

Développement d'un jeu en Java avec interface graphique et réseau

Description du Projet

Contexte

Ce projet a été réalisé dans le cadre du BTS SIO SLAM. Il s'agit d'un jeu multijoueur basé sur une architecture client-serveur en Java. L'application gère une arène graphique dans laquelle les joueurs peuvent interagir, se déplacer, et discuter via un tchat en temps réel.

Objectifs
  • Développer une interface graphique interactive avec Swing
  • Gérer les connexions réseau (clients et serveur)
  • Synchroniser les objets du jeu : murs, personnages, boules
  • Mettre en place un tchat fonctionnel
  • Assurer un bon affichage côté client et serveur
Fonctionnalités Clés
Connexion multi-joueurs via réseau
Tchat en direct intégré à l’arène
Murs dynamiques générés aléatoirement
Synchronisation client/serveur en temps réel
