RP4 - Solution Cloud

Déploiement et gestion d'infrastructure cloud moderne

Description du Projet

Contexte

Cette réalisation professionnelle portait sur la migration et le déploiement d'applications vers le cloud. Le projet incluait la conteneurisation des applications, l'automatisation du déploiement et la mise en place d'une infrastructure évolutive et sécurisée.

Objectifs
  • Migrer des applications legacy vers le cloud
  • Mettre en place une infrastructure as code
  • Automatiser les processus de déploiement
  • Assurer la scalabilité et la haute disponibilité
  • Optimiser les coûts d'infrastructure
Services Déployés
Serveurs Web - Applications conteneurisées
Bases de Données - Services managés RDS
Load Balancer - Répartition de charge
Monitoring - Surveillance et alertes
