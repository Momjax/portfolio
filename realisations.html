RP1 - Classcord

Terminé

Application de messagerie instantanée développée en Java, inspirée de Discord. Communication client-serveur via sockets TCP avec échange de données JSON.

Technologies :
Java Swing JSON TCP Sockets
Fonctionnalités clés :
  • Connexion multi-utilisateurs
  • Messages publics et privés
  • Gestion des statuts (disponible, absent...)

RP2 - NutriFit

Terminé

Application web de suivi nutritionnel permettant de calculer le BMR, rechercher des aliments et générer des profils personnalisés avec export PDF.

Technologies :
HTML5 CSS3 JavaScript Bootstrap
Fonctionnalités clés :
  • Calcul du métabolisme de base
  • Recherche d'aliments via API
  • Génération de graphiques et PDF

RP3 - Jeu d'Arène

Terminé

Jeu multijoueur en Java avec architecture client-serveur. Les joueurs interagissent dans une arène graphique avec chat intégré en temps réel.

Technologies :
Java Swing Sockets Multi-threading
Fonctionnalités clés :
  • Connexion multijoueur temps réel
  • Synchronisation des états
  • Chat intégré dans le jeu
