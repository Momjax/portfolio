1. Contexte

Situation

Ce projet a été réalisé dans le cadre d'un stage / projet professionnel pour l'association culturelle Andalounice, spécialisée dans la promotion de la musique andalouse à Nice. L'objectif était de refondre intégralement le site web existant (V1) pour le moderniser, corriger les informations (coordonnées, adresse, horaires) et améliorer l'expérience utilisateur.

Organisation

Projet développé individuellement avec un interlocuteur côté association. Analyse des besoins, correction du contenu existant, développement de la nouvelle version et mise en ligne.

Problématique

Comment refondre un site web associatif existant pour le rendre plus moderne, plus accessible, avec des informations à jour (contact, localisation, formations) et une meilleure présence en ligne ?

2. Besoin & Objectifs

Objectifs fonctionnels

  • Refonte graphique complète du site (charte, typographie, couleurs)
  • Page Formations avec liste dynamique depuis la BDD
  • Page Événements et Page Galerie
  • Système d'inscription avec panier d'achats
  • Paiement en ligne : PayPal (sandbox) + Stripe (mode test)
  • Formulaire de contact avec envoi d'email via PHPMailer
  • Mise en production sur O2Switch avec guide de déploiement

Contraintes techniques

  • Architecture MVC PHP structurée
  • Base de données MySQL avec PDO (protection injections SQL)
  • Sécurisation : validation, .htaccess, mots de passe hashés
  • Composer pour la gestion des dépendances (Stripe, PayPal, PHPMailer)

6. Difficultés & Solutions

Problème 1 : Informations incorrectes dans la V1

L'ancienne version contenait des erreurs partout : mauvais numéro, mauvaise adresse, fautes de frappe.

Solution :

Audit complet du site, création d'un fichier config.php centralisé pour éviter les incohérences futures.

Problème 2 : Intégration Google Maps

L'embed Maps de la V1 ne pointait pas vers la bonne adresse.

Solution :

Génération d'un nouvel embed iframe Google Maps avec les coordonnées GPS exactes de l'association.
